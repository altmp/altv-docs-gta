# Manual loading & unloading

## Blocking

```js
import * as alt from 'alt-client';
// Load the IPL
alt.requestIpl(iplName);
// Unload the IPL
alt.removeIpl(iplName);
```

## Non blocking

```js
import * as game from 'natives';
// Load the IPL
game.requestIpl(iplName);
// Unload the IPL
game.removeIpl(iplName);
```
