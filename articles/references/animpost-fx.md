# Animpost FX

Animpost FX can be used to draw various screen effects to the game camera. These effects can be activated using the [animpostfxPlay](https://natives.altv.mp/0x2206BF9A37B7F724) native function.

## Preview

> [!TIP]
> All images are automatically generated. Screenshot was taken when the progress of animpost FX reaches 100%. However, for effects that end with **"Out"**, screenshot is taken earlier, approximately when the progress is at 30%.
